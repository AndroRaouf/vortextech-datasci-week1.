# Sales Data Analysis

## Project Overview

This project demonstrates the basic process of data cleaning and visualization using Python.

The analysis was performed on a sales dataset containing more than 2,800 records.

## Tools Used

- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

## Project Tasks

- Load the dataset
- Explore the dataset
- Handle missing values
- Remove duplicate records
- Check data types
- Create meaningful visualizations

## Visualizations

- Orders by Status
- Top 10 Countries by Sales
- Sales Distribution
- Total Sales by Year
- Sales Distribution by Deal Size
